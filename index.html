{%- comment -%} An toàn: nếu site.issues nil thì lấy từ site.collections['issues'].docs, nếu vẫn không có thì gán mảng rỗng để tránh lỗi sort. {%- endcomment -%} {%- assign _issuesA = site.issues -%} {%- if _issuesA == nil or _issuesA == empty -%} {%- assign _issuesColl = site.collections['issues'] -%} {%- if _issuesColl and _issuesColl.docs -%} {%- assign _issuesA = _issuesColl.docs -%} {%- else -%} {%- assign _issuesA = "" | split: "," -%} {%- endif -%} {%- endif -%} {%- assign issues_sorted = _issuesA | sort: 'date' | reverse -%} {%- for post in issues_sorted -%} {{ post.title }} {%- endfor -%} {%- if issues_sorted == empty -%} Chưa có bài nào trong _issues/. Thêm file để bắt đầu. {%- endif -%}

Cách dùng

  • Mỗi số (tháng/tuần) là một file trong thư mục _issues/ (ví dụ: 2025-09.md, 2025-10-w1.md).
  • Front-matter cần có: title, date, layout: week, type: month hoặc type: week.
  • Trang này tự tạo danh sách “chip” theo thời gian (mới nhất trước), không cần sửa code.
